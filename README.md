# Unlox Web Development

This workspace contains a set of mini projects from Week 1 to Week 6, plus one major project: Outpro.India.

## Projects at a Glance

### Week 1 - Finance Dashboard
A static frontend dashboard focused on financial data presentation. It is a mini project built with HTML and CSS to practice layout structure, cards, and visual hierarchy.

### Week 2 - TechGadget Landing Page
A responsive product landing page for a technology brand. This mini project is centered on marketing design, hero sections, call-to-action placement, and polished presentation.

### Week 3 - Weather and Movie Search App
A small interactive web app that combines weather and movie search experiences. This mini project is meant to practice JavaScript logic, API-style UI flows, and user interaction.

### Week 4 - Desk: To-Do App + Notes App
A React mini project that combines a task manager and a notes app in one interface. It demonstrates component-based UI, local state management, and a tab-based experience.

### Week 5 - E-Commerce Frontend
A React shopping frontend built with routing and cart context. This mini project includes product listing, product details, cart management, and cart persistence in local storage.

### Week 6 - Blog REST API
A Node.js and Express REST API for blog posts. This mini project covers CRUD operations, authentication, validation, JSON file storage, and API documentation.

### Outpro.India - Major Project
Outpro.India is the major project in this workspace. It is a Next.js 14 corporate website built for a real business-style brief, with multiple pages, reusable components, backend route handling, MongoDB integration, email handling, SEO routes, and a more complete production-ready structure.
